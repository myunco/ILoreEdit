# ILoreEdit
<span>[ 中文 | <a href="README.md">English</a> ]</span>

ILoreEdit 是一个 Minecraft 服务器插件，提供编辑物品显示名称和Lore、修改头颅主人以及设置自定义模型数据等功能。

有关插件的详细说明，请访问 ~~[mcbbs.net](https://www.mcbbs.net/thread-1160634-1-1.html)~~ [myunco.net](https://myunco.net/thread-3-1-1.html).

如何安装？
---
* 下载最新版本插件。
* 将 ILoreEdit-1.x.x.jar 文件放入 plugins 文件夹中。
* 启动服务器。 (支持Bukkit/Spigot/Paper及衍生服务端)
* 现在插件应该在 /plugins 命令中显示为绿色。 如果显示红色或不存在，请检查控制台日志的错误信息。

构建
---
克隆此存储库后，将 [Craftbukkit-1.14.4.jar](https://getbukkit.org/get/CiNKyh4l9MuPHLpovnGSDU2oHT9gCpUc) 和 lib 目录添加到依赖项，使用 IDEA Artifacts 构建项目。
